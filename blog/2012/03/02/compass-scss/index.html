Scss开发利器Compass

Mar 2nd, 2012

一.何为Compass

Compass is an open-source CSS Authoring Framework.

Compass是一个开源的CSS创作框架。

Compass = Sass framework powered by community

Compass特点

  • 拥有内建的CSS3 mixins
  • 很多的实用的CSS取代方法
  • 很多辅助工具
  • 输出SASS文件为压缩的CSS

二.安装Compass建立项目

1.安装Compass Compass需要运行在Ruby环境下，必须确定本地已经安装Ruby。

2.建立项目

在config.rb配置文件中，可以修改一些基础配置

3.实时编译Scss文件

每次保存完scss文件，compass会实时编译生成css文件。

三.Compass核心框架

1.CSS3

说实话开始我很厌烦CSS3的原因在于我要花费很多时间来处理不同浏览器属性，如：-webkit， -moz等等。 CSS3圆角

CSS3的渐变色

面对这样繁琐的浏览器属性谁的头都会大，看看Compass是怎样一句话解决的。

在官网上也有很多渐变的Demo，还有圆角等等。

更多的Compass CSS3请到官方网站。

你还在厌烦CSS3不同浏览器属性吗？快来用Compass吧！

2.Typography （文字基础样式）

  • Links
  • Lists
  • Text
  • Vertical Rhythm

官网有详细的文档

3.Utilities（实用工具）

  • Color
  • General
  • Print
  • Sprites
  • Tables

里面有个最实用的功能当属sprite。 你在还在一点一点的拼sprite图吗？快用Compass吧。
我们只要把小图放在一个文件夹里，compass会自动合并并计算出位置。

compass生成后的CSS，强的一米。

4.Reset

一句话把所有的reset搞定

5.Layout

Compass也提供layout布局，简单看了，貌似用处不大

四.Blueprint Module

Compass里面带有Blueprint框架，在建立项目的时候可以直接应用到项目中。

向已建立的项目中加入Blueprint

具体的试用可以到官网上去查询。

五.其他

你可以去github上引用他人已经做好的效果，比如 Fancy Buttons
