从零开始在github上安装octopress

Jan 18th, 2012

经过我不懈的努力，终于把Blog搞定了，通过这次折腾，学到了很多东西

在octopress上面有具体的文档，E文好的可以直接，[点击查看 http://octopress.org/docs/]

Octopress 安装

需要的环境

* git
* ruby 1.9.2

安装Git

首页git是必备的，具体看到github官网上 http://help.github.com/mac-set-up-git/，这个就比较简单了。

安装Ruby

Ruby可以利用RVM或者rbenv来安装，很轻松帮你搞定首页Ruby环境，我是用RVM来安装的。

1.安装 RVM

RVM(Ruby Version Manager)

$ bash -s stable < <(curl -s https://raw.github.com/wayneeseguin/rvm/master/binscripts/rvm-installer)
2.安装Ruby1.9.2
rvm install 1.9.2 && rvm use 1.9.2
3.安装Octopress
git clone git://github.com/imathis/octopress.git octopress
cd octopress    //这边会有提示信息，yes就行
ruby --version  //Ruby的版本需要在1.9.2版本
4.安装环境插件等等
gem install bundler
bundle install
5.安装Octopress 主题
rake install
