2011年总结及2012年计划

Jan 3rd, 2012

2011总结

一.生活

  1. 去年最大的事就是结婚了，由男孩成长为男人，责任更大了，压力更大了。

  2. 终于狠心在亚马逊上面分期购买了MBP，为抢到亚马逊的秒杀价格特意写了一个小的JS程序，后来为了更好的体验，升级了SSD和8G内存，最后年底也入手了magic mouse，烧钱啊。

  3. 旅游
    • 浙江安吉
    • 杭州瘦西湖
    • 帝都
  1. 老婆在怀孕了，来年我们将有个龙宝宝了（^ - ^）。
  2. 2012年元旦是我们结婚一周年，特意用一周的时间做了一个周年纪念视频。

二.工作及技术成长

  1. 前端团队的建设以及前端基础架构规范文档知识梳理
  2. Mobile web的研究试水
  3. 开始IOS学习，渐渐入门
  4. 熟悉OSX下的各种开发环境及工具
  5. 部门分享
    • 前端开发基础分享
    • Mobile web 开发实践分享

三.读书

  1. 《乔布斯传记》
  2. 《Object-C》
  3. 一堆分享的PDF

2012计划

  1. 加强学习IOS开发，多学技术。
  2. 房子和车子不变的话题，年中孩子就要出生了，这两个最重要的问题要解决
  3. 两地分居的问题，争取在年初解决
